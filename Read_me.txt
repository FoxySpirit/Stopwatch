Спортивный секундомер для отсчёта кругов у 4-х спортсменов. Изначально задумывался для автоспорта (картинг, автокросс).
Секундомер дорабатывается и на сей момент функционал составляет:
- Раздельный отсчёт времени круга с раздельным стартом
- Отображение 5-и последних кругов
- Лучшее время спортсмена
- Построение графика по времени для оценки стабильности

Порядок использования: 
-открытие файла SW.html в браузере
-нажатие кнопки "set" для установки стартовых номеров
(стартовые номера прописываются во всплывающих окнах)
-отсчёт кругов нажатием на окошко с соответствующим стартовым номером
-нажатие кнопки "Stop" для остановки секундомера
-последующее нажатие кнопки "graph" для вывода временного графика по каждому спортсмену
(При выводе графика на нём отображается синим квадратиком лучщее время)
